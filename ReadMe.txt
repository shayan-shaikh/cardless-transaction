No requirements of installing any additional softwares.
Only requirements are :
any Python IDE
A stable internet (for downloading libraries)
----------------------------------------------------------------------------------------------------------------------
List of Python libraries used :
tkinter
PIL
qrcode
resizeImage
kivy
kivymd
opencv (cv2)
numpy
subprocess (in built library)
random (in-built library)
pandas
fpdf
datetime (in-built)
-----------------------------------------------------------------------------------------------------------------------------
To run the app, firstly install Python latest stable version (3.10) from the official website
Then run the following commands to install the libraries :
pip install tkinter
pip install qrcode
pip install pillow
pip install kivy
pip install kivymd
pip install numpy
pip install opencv
pip install fpdf
----------------------------------------------------------------------------------------------------------------------------------
After it is done, load the files in an IDE.
Wherever any object references the location, change it to the location which is in the host device.
Run the programs, everything should work fine.
